## Instagram

### Copy del post (descripción / caption)
```
El revenge trading no se siente como un error cuando lo estás cometiendo.

Se siente como oportunidad.

El patrón:
→ Pérdida grande
→ "Tengo que recuperarme"
→ Trade apresurado fuera del plan
→ Pérdida mayor
→ Evaluación en riesgo

3 preguntas de autodiagnóstico:
¿Tomaste un trade en los 30 minutos siguientes a una pérdida grande?
¿El tamaño de posición fue mayor de lo normal?
¿Estabas siguiendo tu plan o tratando de recuperar?

La solución no es fuerza de voluntad.
Es tener datos que te lo muestren antes de que el daño esté hecho.

Guarda este post. La próxima vez que sientas ese impulso, vuelve a leerlo.
```
